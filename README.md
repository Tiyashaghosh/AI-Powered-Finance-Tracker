# AI Finance Tracker

A full stack AI-powered personal finance tracking application
that helps users track income and expenses with automatic
ML-based categorization and an LLM-powered finance chatbot.

## Tech Stack
- Backend: FastAPI, PostgreSQL, SQLAlchemy
- ML: scikit-learn, TF-IDF, Logistic Regression
- LLM: OpenAI API
- Frontend: React, Tailwind CSS, Recharts
- DevOps: Docker, Render, Vercel

## Features (building in progress)
- [ ] User authentication (JWT)
- [ ] Transaction CRUD
- [ ] ML-powered auto categorization
- [ ] LLM finance chatbot
- [ ] Spending analytics dashboard
